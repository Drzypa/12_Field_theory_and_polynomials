proof of the fundamental theorem of algebra (Liouville’s theorem)


Let f:ℂ→ℂ be a polynomial, and suppose f has no root in ℂ. We will show f is constant.

Let g=1f. Since f is never zero, g is defined and holomorphic on ℂ (ie. it is entire). Moreover, since f is a polynomial, |f⁢(z)|→∞ as |z|→∞, and so |g⁢(z)|→0 as |z|→∞. Then there is some M>0 such that |g⁢(z)|<1 whenever |z|>M, and g is continuous and so bounded on the compact set {z∈ℂ:|z|≤M}.

So g is bounded and entire, and therefore by Liouville’s theorem g is constant. So f is constant as required.□
