algebraic extension


Definition 1.

Let L/K be an extension of fields. L/K is said to be an algebraic extension of fields if every element of L is algebraic over K. If L/K is not algebraic then we say that it is a transcendental extension of fields.

Examples:

  1. 1.

    Let L=ℚ⁢(2). The extension L/ℚ is an algebraic extension. Indeed, any element α∈L is of the form

    α=q+t⁢2∈L

    for some q,t∈ℚ. Then α∈L is a root of

    X2-2⁢q⁢X+q2-2⁢t2=0
  2. 2.

    The field extension ℝ/ℚ is not an algebraic extension. For example, π∈ℝ is a transcendental number over ℚ (see pi). So ℝ/ℚ is a transcendental extension of fields.

  3. 3.

    Let K be a field and denote by K¯ the algebraic closure of K. Then the extension K¯/K is algebraic.

  4. 4.

    In general, a finite extension of fields is an algebraic extension. However, the converse is not true. The extension ℚ¯/ℚ is far from finite.

  5. 5.

    The extension ℚ⁢(π)/ℚ is transcendental because π is a transcendental number, i.e. π is not the root of any polynomial p⁢(x)∈ℚ⁢[x].
