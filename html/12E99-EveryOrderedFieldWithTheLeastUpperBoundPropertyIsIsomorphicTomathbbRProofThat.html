every ordered field with the least upper bound property is isomorphic to ℝ, proof that


Let F be an ordered field with the least upper bound property. By the order properties of F, 0<1F and by an induction argument 0<n⋅1F for any positive integer n. Hence the characteristic of the field F is zero, implying that there is an order-preserving embedding j:ℚ→F.

We would like to extend this map to an embedding of ℝ into F. Let r∈ℝ and let Dr={q∈ℚ:q<r} be the associated Dedekind cut. Since Dr is nonempty and bounded above in ℚ, it follows that the set j⁢(Dr) is nonempty and bounded above in F. Applying the least upper bound property of F, define a function ȷ~:ℝ→F by

ȷ~⁢(r)=sup⁡(j⁢(Dr)).

One can check that ȷ~ is an order-preserving field homomorphism. By replacing F with the isomorphic field F∖ȷ~⁢(ℝ)∪ℝ, we may assume that ℝ⊂F.

We claim that in fact ℝ=F. To see this, first recall that since F is a partially ordered group with the least upper bound property, F has the Archimedean property (http://planetmath.org/DistributivityInPoGroups). So for any f∈F, there exists some positive integer n such that -n<f<n. Hence the set Df′={r∈ℝ:r<f}⊂ℝ is nonempty and bounded above, implying that f′=supℝ⁡Df′ lies in ℝ. Now observe that applying the least upper bound axiom in F gives us that f=supF⁡Df′. Since f′ is an upper bound of Df′ in F, it follows that f≤f′.

Seeking a contradiction, suppose f<f′. By the Archimedean property, there is some positive integer n such that f<f′-n-1<f′. Because f′=supℝ⁡Df′, we obtain f′-n-1<f, which implies the contradiction f<f. Therefore f=f′, and so f∈ℝ. This completes the proof.
