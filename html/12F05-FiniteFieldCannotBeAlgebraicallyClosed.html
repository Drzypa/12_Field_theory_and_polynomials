finite field cannot be algebraically closed


Theorem.

A finite field cannot be algebraically closed.

Proof.

The proof proceeds by the method of contradiction. Assume that a field F is both finite and algebraically closed. Consider the polynomial p⁢(x)=x2-x as a function from F to F. There are two elements which any field (in particular, F) must have — the additive identity 0 and the multiplicative identity 1. The polynomial p maps both of these elements to 0. Since F is finite and the function p:F→F is not one-to-one, the function cannot map onto F either, so there must exist an element a of F such that x2-x≠a for all x∈F. In other words, the polynomial x2-x-a has no root in F, so F could not be algebraically closed. ∎
