equivalent statements of Lindemann-Weierstrass theorem


Proposition 1.

The following versions of the Lindemann-Weierstrass Theorem are equivalent:

  1. 1.

    If α1,…,αn are linearly independent algebraic numbers over ℚ, then eα1,…,eαn are algebraically independent over ℚ.

  2. 2.

    If α1,…,αn are distinct algebraic numbers over ℚ, then eα1,…,eαn are linearly independent over ℚ.

Proof.

(1⟹2). Write Ai=eαi for each i=1,…,n. Suppose 0=r1⁢A1+⋯+rn⁢An, where ri∈ℚ. Moving r1⁢A to the LHS and by multiplying a common denominator we can assume that r1⁢A1=r2⁢A2+⋯+rn⁢An where ri∈ℤ. We want to show that r1=⋯=rn=0. We induct on n. The case when n=1 is trivial because A1 is never 0 and therefore 0=r1⁢A1 forces r1=0.

By induction hypothesis, suppose the statement is true when n<k. Now suppose n=k. If α1,…,αk are linearly independent over ℚ, A1,…,Ak are algebraically independent and certainly linearly independent over ℚ. So suppose α1,…,αk are not linearly independent over ℚ. Without loss of generality, we can assume s1⁢α1=s2⁢α2+⋯+sk⁢αk, where si∈ℚ and s1≠0. By multiplying a common denominator we can further assume that si∈ℤ and s1>0. Then

A1s1=es1⁢α1=es2⁢α2+⋯+sk⁢αk=es2⁢α2⁢⋯⁢esk⁢αk=A2s2⁢⋯⁢Aksk.

Since r1⁢A1=r2⁢A2+⋯+rn⁢An, we get

(r1s1)⁢(A2s2⁢⋯⁢Aksk) = (r1s1)⁢A1s1
= (r1⁢A1)s1
= (r2⁢A2+⋯+rk⁢Ak)s1
= g⁢(A2,…,Ak),

where g⁢(x2,…,xk)=(r2⁢x2+⋯+rk⁢xk)s1∈ℚ⁢[x2,…,xk]. Partition the numbers si′⁢s into non-negative and negative ones, so that, say, sc⁢(1),…,sc⁢(l) are non-negative and sd⁢(1),…,sd⁢(m) are negative, then

(r1s1)⁢Ac⁢(1)sc⁢(1)⁢⋯⁢Ac⁢(i)sc⁢(i)=g⁢(A2,…,Ak)⁢Ad⁢(1)sd⁢(1)⁢⋯⁢Ad⁢(j)sd⁢(j).

If we define

f⁢(x2,…,xk)=r1s1⁢xc⁢(1)sc⁢(1)⁢⋯⁢xc⁢(i)sc⁢(i)-g⁢(x2,…,xk)⁢xd⁢(1)sd⁢(1)⁢⋯⁢xd⁢(j)sd⁢(j),

then f⁢(x2,…,xk)∈ℚ⁢[x2,…,xk] and f⁢(A2,…,Ak)=0. By the induction hypothesis, f=0. It is not hard to see that r1=⋯=rk=0 and therefore A1,…,Ak are linearly independent.

(1⟸2). We first need two lemmas:

Lemma 1. Given 2., if α≠0 is algebraic over ℚ, then eα is transcendental over ℚ.

Proof.

Suppose f⁢(eα)=0 where f⁢(x)=r0+r1⁢x+⋯+rn⁢xn∈ℚ⁢[x]. Then we have

0 = r0+r1⁢eα+⋯+rn⁢(eα)n
= r0⁢e0+r1⁢eα+⋯+rn⁢en⁢α.

Since α≠0, 0,α,…,n⁢α are all distinct, 1,eα,…,en⁢α are linearly independent by the hypothesis. Thus, r0=r1=…=rn=0 and we have f⁢(x)=0, which means that eα is transcendental over ℚ. ∎

Lemma 2. Given 2., if α and β are linearly independent and algebraic over ℚ, then eα is transcendental over ℚ⁢(eβ).

Proof.

Let A=eα and B=eβ. Suppose f⁢(A)=0 where f⁢(x)∈ℚ⁢(B)⁢[x]. We want to show that f⁢(x)=0. Write

f⁢(x)=r0⁢(B)+r1⁢(B)⁢x+⋯+rn⁢(B)⁢xn,

where each ri⁢(x)=pi⁢(x)/qi⁢(x) with pi⁢(x), qi⁢(x)≠0∈ℚ⁢[x]. Let Q⁢(x)=q1⁢(x)⁢⋯⁢qn⁢(x). So Q⁢(B), being the product of the denominators qi⁢(B)≠0, is non-zero. Multiply f⁢(x) by Q⁢(B) we get a new polynomial g⁢(x) such that

g⁢(x)=R0⁢(B)+R1⁢(B)⁢x+⋯+Rn⁢(B)⁢xn,

where each Ri⁢(x)=ri⁢(x)⁢Q⁢(x)=pi⁢(x)⁢Q⁢(x)/qi⁢(x)∈ℚ⁢[x]. Now, g⁢(A)=f⁢(A)⁢Q⁢(B)=0. So

0 = R0⁢(B)+R1⁢(B)⁢A+⋯+rn⁢(B)⁢An
= ∑j=0m0a0⁢j⁢Bj+∑j=0m1a1⁢j⁢Bj⁢A+⋯+∑j=0mnan⁢j⁢Bj⁢An
= ∑j=0m0a0⁢j⁢ej⁢β+∑j=0m1a1⁢j⁢ej⁢β+α+⋯+∑j=0mnan⁢j⁢ej⁢β+n⁢α,

where each ai⁢j∈ℚ. Now, the exponents in the above equation are all distinct, or else we would end up with α and β being linearly dependent, contrary to the assumption. Therefore, by 2 (Lindemann-Weierstrass Version 2), all ei⁢β+j⁢α are linearly independent, which means each ai⁢j=0. This implies that g⁢(x)=0. But g⁢(x)=f⁢(x)⁢Q⁢(B) and Q⁢(B)≠0, we must have f⁢(x)=0. ∎

Now onto the main problem. We proceed by induction on the number of linearly independent algebraic elements over ℚ. The case when n=1 is covered in Lemma 1, since a linearly independent singleton is necessarily non-zero. So suppose α1,…,αk are linearly independent and algebraic over ℚ. Then each pair αk,αi are independent and algebraic over ℚ, i≠k. Thus eαk is transcendental over ℚ⁢(eαi) for all i≠k. This means that eαk is transcendental over ℚ⁢(eα1,…,eαk-1).

Now let Ai=eαi for all i=1,…,k. Suppose f⁢(A1,…,Ak)=0 where f∈ℚ⁢[x1,…,xk]. To show the algebraic independence of the Ai′⁢s, we need to show that f=0. Rearranging terms of f and we have

0=f⁢(A1,…,Ak)=∑j=0mgj⁢(A1,…,Ak-1)⁢(Ak)j.

If we let g⁢(x)=f⁢(A1,…,Ak-1,x), we see that g⁢(x)∈ℚ⁢(A1,…,Ak-1)⁢[x] and g⁢(Ak)=f⁢(A1,…,Ak-1,Ak)=0. Since eαk is transcendental over ℚ⁢(A1,…,Ak-1), we must have g⁢(x)=0. This implies that each gj⁢(A1,…,Ak-1)=0. But then A1,…,Ak-1 are algebraically independent by the induction hypothesis, we must have each gj=0. This means that f=0. ∎
